# MLflow Learning Repository

This repository contains hands-on experiments and mini-projects to understand **MLflow for experiment tracking, model evaluation, and monitoring**. It demonstrates how MLflow can be used in real-world machine learning workflows, from training models to tracking performance and detecting drift.

---

## Objectives

* Understand MLflow experiment tracking
* Compare multiple model runs
* Perform hyperparameter tuning
* Monitor model performance
* Simulate model drift
* Build dashboards for visualization

---

## Tech Stack

* Python
* MLflow
* Scikit-learn
* Pandas
* Streamlit
* Matplotlib

---

## MLflow Features:

* Experiment tracking
* Parameter logging
* Metric logging
* Model logging
* Run comparison
* Model performance monitoring
